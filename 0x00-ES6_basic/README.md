## Introduction

ECMAScript 6 (ES6) introduced many new features to JavaScript, making it more powerful and easier to write. This project aims to familiarize you with these new features through practical tasks and exercises.

## Learning Objectives
By the end of this project, you should be able to:

* Understand the new let and const keywords for variable declaration
* Use arrow functions for concise function syntax
* Utilize template literals for string interpolation
* Implement destructuring for arrays and objects
* Understand and use default parameters in functions
* Work with the rest and spread operators
* Understand block-scoping and the let and const keywords
* Use ES6 modules with import and export

## Requirements

* Node.js installed on your local machine
* A code editor (e.g., Visual Studio Code)
* Basic understanding of JavaScript
